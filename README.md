# hiviewdfx_api_metrics

API Metrics SIG仓，提供支持API直方图高性能低功耗打点功能，提供了计数、枚举、时间、百分比、布尔五种直方图

# 目录
  hiviewdfx_api_metrics
├─config/                 # 项目配置
├─document/               # 项目文档
├─resources/              # 依赖库
├─src/
│　├─checker/             # 项目检测规则功能代码
│　├─codeFix/             # 修复
│　├─matcher/             # 匹配类型
│　├─model/               # 模块
│　├─utils/               # 公共接口
│　└─run.ts               # 项目入口
└─test/                   # 测试目录
